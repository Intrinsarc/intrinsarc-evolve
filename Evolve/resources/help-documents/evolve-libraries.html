The following third party libraries and icon sets are used in Evolve:

The Jazz 2D Scenegraph, licensed under the MPL
Jazz is a fast and flexible 2D toolkit for displaying large scenegraphs of shapes on a Swing canvas. Many thanks to Ben Bederson and the developers at HCiL Maryland.

Eclipse UML2, licensed under the EPL
Eclipse UML2 is complete and pragmatic implementation of the UML2 metamodel. Kudos to the UML2/EMF developers for an astonishingly good toolkit.

Docking Frames, licensed under the LGPL
This is a feature rich Swing docking framework. Thanks to Benjamin Sigg for the excellent support and hard work.

ASM, licensed under the INRIA license
ASM is an exceedingly fast and flexibly bytecode reading and manipulating framework. It is used by the bean importer.

JTattoo Look and Feel, licensed commercially
JTattoo is a set of clean Swing Look and Feel themes. Thanks for the excellent work and timely bug fixes Michael.

FreeHEP Vector Graphics, licensed under the LGPL
This comprehensive vector graphics library is used for exporting diagrams in Windows EMF format or EPS format.

JSyntaxPane, licensed under the ASL2.0
JSyntaxPane is a robust syntax highlighting Swing component, used to display Backbone code.

FAMFAMFAM Silk Icons, licensed under the Creative Commons Attribution license
Many of the 16x16 icons in Evolve are from this set. Thankyou to Mark James for the stylish work.

L2FProd Common Components, licensed under the ASL2.0
This is used for choosing directories.

Xoetrope Color Wheel, licensed under the XUI license
The Xoetrope color wheel is a functional and elegant color chooser.

Ekit, licensed under the LGPL
Ekit provides a feature-rich HTML editor, used for HTML notes in Evolve.

Sulky memory monitor, licensed under the LGPL and ASL2.0
Sulky memory monitor is a Swing component, written by Joern Huxhorn, which shows how much memory is allocated and used.

Print preview classes, written by Alno Dice

JIcon © Claude Duguay 2001-2002
This is used to overlay icons in the repository browser.
